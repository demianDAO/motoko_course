# motoko_course

技术栈: `Motoko + Rust + React + Typescript + Tailwind2`

学完本课程你将获得:

- [x] 掌握 ICP 的基础
- [x] 理解 ICP 的经济模型和运行原理
- [x] 基于 Motoko 可以开发属于自己的 dApp
- [x] 从 0 到 1 成为独立开发者

## 课程规划

一共分为三个主要阶段和两个探索阶段, 探索是为了更好的实现合约和编写代码.
